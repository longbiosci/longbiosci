# LongBiosci 内容模板

## 新闻文章模板

```
标题：核心发现（一句话说明）
标签：研究进展/融资/临床/突破/政策
来源：[来源网站名称](来源URL)
日期：YYYY-MM-DD

开头：一句话背景介绍

核心发现：
- 关键点1
- 关键点2  
- 关键点3

作用机制：（如有）

研究意义：（如有）

参考文献/链接：[来源](URL)
```

## 示例

```markdown
标题：肠道细菌 Roseburia inulinivorans 增加肌肉力量
标签：研究进展
来源：[Fight Aging!](https://www.fightaging.org/)
日期：2026-03-17

研究表明，肠道微生物组随着年龄增长而发生变化。近期研究发现特定细菌种类与肌肉力量之间存在显著相关性。

核心发现：
- 细菌种类: Roseburia inulinivorans
- 关联: 与握力、腿举、胸推力量呈正相关
- 效果: 补充后小鼠肌肉力量提升约30%
- 人群差异: 老年人体内该细菌丰度低于年轻人

作用机制：
- 降低盲肠和血浆中的氨基酸浓度
- 激活肌肉中的嘌呤和磷酸戊糖途径
- 增加肌纤维尺寸

研究意义：
- 单一菌种益生菌生产成本低
- 有望成为治疗年龄相关性肌肉萎缩的候选方案
- 为"肠-肌轴"理论提供新证据

参考文献: [Gut Journal](https://doi.org/10.1136/gutjnl-2025-336980)
```
